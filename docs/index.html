Intoduction

This is the documentation for UK Police Memorial API Service, it outlines out the use the API Connectors.
To use this API you require are public and secret key from the UKPM Trust.

This document provides examples for PHP & Javascript if you would like to hide the languages your not using. You can change the display at the bottom of the left hand menu.

Connectors

The connectors are a set of libraries allowing quick and easy communication with the UKPM API Service.

PHP Connector

The php connectors are a objective library that can be installed via Composor BASH / CMD

composer require ukpm/php_connector

Once installed require the composor vendor "autoload.php" PHP

require 'vendor/autoload.php';

This then allows you to use the library PHP

use UKPM\Connector;
use UKPM\Client;

$publicKey = "" /* This should be your public key */;
$secretKey = "" /* This should be your secret key */;

$connector = new Connector(new Client($publicKey, $secretKey), true);

JavaScript V1 Connector

The JavaScript V1 connector HTML

<script src="https://cdn.jsdelivr.net/npm/promise-polyfill@7/dist/polyfill.min.js"></script>
<script type="text/javascript" src="js/ukpm.js"></script>
<script type="text/javascript" src="js/sodium.js"></script>
<script type="text/javascript">
(async() => {
	await sodium.ready;
	var client_key = "" /* This should be your public key */;
	var client_secret = "" /* This should be your secret key */;

	var client = new window.UKPM(
		client_key,
		client_secret
	);
})();
</script>

Methods

Test

This endpoint group is to enable developers to quickly and easily test their connection to the UKPM API Service

test

Provides developers a test case for handling a response from the API Server.

  PHP

$output = $connector->test->get();
print_r($output);

  Javascript v1

client.post('test/test', {}).then(function(result){
	console.log("Success");
	console.log(result);
}).catch(function(err){
	console.log("failure");
	console.log(err);
});

  Javascript v2

client.test.Test().then(function(result){
	console.log("Success");
	console.log(result);
}).catch(function(err){
	console.log("failure");
	console.log(err);
});

Wall

Allows access to a set of random profile picutes.

Get

Provides a list of 40 random profile pictures.

  PHP

$connector->wall->get();

  Javascript v1

client.post('wall/get', {});

  Javascript v2

client.wall.Get();

Officer

Allows access information specific to a single officer.

Get

Provides an officer's information and story if it's exists within the system.

  PHP

$connector->officer->get($officer_id);

  Javascript v1

client.post('officer/get', {"officer_id": officer_id});

  Javascript v2

client.officer.Get(officer_id);

Ranks

List

  Javascript v2

client.ranks.List();

Officers

Allows access to all officers registered in the system.

ByForce

Provides a list of all officers that died in the given force.

  PHP

$connector->officers->getByForce($force_id);

  Javascript v1

client.post('officers/force', {"force_id": force_id});

  Javascript v2

client.officers.Force(force_id);

byRank

Provides a list of all officers that died within the given rank

  PHP

$connector->officers->getByRank($rank_id);

  Javascript v1

client.post('officers/rank', {"rank_id": rank_id});

  Javascript v2

client.officers.Rank(rank_id);

BySearchTerm

Provides a list of all officers that match on the given search term.

  PHP

$connector->officers->getBySearchTerm($searchTerm);

  Javascript v1

client.post('officers/search', {"search_term": search_term});

  Javascript v2

client.officers.search(search_term);

ByDayInMonth

Provides a list of all officers who died on the given day/month.

  PHP

$connector->officers->getByDayInMonth($day, $month);

  Javascript v1

client.post('officers/day', {"day": 01, "month":01});

  Javascript v2

client.officers.Day(day, month);
